# Airp_sh
###### Airp的一键脚本仓库
#### Ap-Box----多功能工具箱 注意:本脚本暂时只适配amd64和X86平台 暂时arm机器未适配
```
wget -O apbox.sh https://raw.githubusercontent.com/Aorp2008/Airp_sh/main/apbox.sh && chmod +x apbox.sh && clear && ./apbox.sh
```
